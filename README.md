# Base Turizm - Vercel Next.js Site

Kurulum:

```bash
npm install
npm run dev
```

Vercel yayına alma:

1. Projeyi GitHub'a yükleyin.
2. Vercel > Add New Project > GitHub repo seçin.
3. Framework: Next.js
4. Build command: `npm run build`
5. Output otomatik algılanır.

İçerikler `app` klasörü içindedir. Telefon/adres bilgileri `components/SiteHeader.tsx` ve `app/iletisim/page.tsx` içinde düzenlenebilir.
